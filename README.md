## Python-Based Pencari Hadis untuk Termux
Ini project scraping pertama saya.
Menggunakan API dari carihadis.com

## Penggunaan
quick start:
```bash
$ pkg install python 
$ pip install bs4
$ cd && git clone https://github.com/annzc/cari-hadis.git
$ cd cari-hadis
$ chmod +x carihadis
$ ./carihadis
```
